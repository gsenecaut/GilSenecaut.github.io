R

Satellite Analyses

This project was a group assignment for the final assessment of the Statistical Learning course in my Master's where we chose to look at a dataset of publicly known satellites in orbit and statistical techniques in R for its analysis. The dataset and variables are introduced, followed by dimension reduction techniques to extract important information. Classification methods like clustering and regression are employed to classify satellites based on their characteristics. The project aims to identify patterns and structures in the data as well as provide insights into satellite classification.

Data Cleaning

This SQL project involves cleaning and standardizing data from a Nashville housing dataset. The script utilizes various data cleaning techniques such as standardizing date formats, filling missing data using related records, and breaking out address data. It also removes duplicate records and unused columns, with the goal of improving data quality and consistency for further analysis.

SQL

Data Exploration

This SQL project aims to explore COVID-19 data using SQL queries. The project begins by importing a dataset containing information on COVID-19 cases, deaths and vaccinations across the world. The script creates various views including information such as the likelihood of death if an individual contracts Covid-19 in a given country and the number of people that have contracted Covid-19 as a percentage of the population in a given location. Overall, this script provides views to be used for visualisations in the Tableau project to allow for a comprehensive analysis of Covid-19 data.
